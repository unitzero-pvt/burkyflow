# Cold Email Variants by Angle — UNITZERO

**Client:** https://unitzero.tech
**Date compiled:** 2026-04-29
**Chains from:** [01](../icp%20unitzero/01_context_and_data_source.md), [02](../icp%20unitzero/02_pain_points_and_consequences.md), [03](03_outcomes_and_whys.md), [04](04_email_copies.md).

**Important note on offers.** The 4 offer descriptions in the brief (A/B/C/D) are *angles*, not scripted offer copy. The variants below treat each as a strategic angle and re-frame the mechanism, the risk-reversal, and the price math in fresh language for each variant — no reuse of the exact "$50/booked job," "$497 pilot," "$2K + $1.5K/mo," or "free 14-day pilot" promise wording. Reps customize the actual commercial terms during the call.

**Variant doctrine.** Each angle gets **4 fresh variants**, each one differs from the others on at least 3 of these 5 axes:
1. **Hook type** — number / quote / question / observation / contrarian.
2. **Pattern interrupt** — verbatim peer phrase / hyper-specific time / competitor name / odd-format subject / negative-CTA.
3. **Mechanism framing** — list reactivation / paid funnel / outbound engine / coverage backstop.
4. **CTA type** — numbered reply / one-question / "tell me to stop" / preview-the-list / day-X promise.
5. **5th-Why hit** — felt-dumb / sleep-at-night / fragile-stack / customer-perception / one-team / identity.

All emails ≤95 words. Every email passes [sops/qa_checklists.md](../../../sops/qa_checklists.md).

## QA against [sops/qa_checklists.md](../../../sops/qa_checklists.md)

- ✅ Why them / why now: city + leaf + season/event/quote in every email.
- ✅ Industry lingo: ServiceTitan, Dentrix, Follow Up Boss/kvCORE, LSAs, MLS, Apollo, Clay, GHL, conflict-check, recall list, dispatch board, expired listings.
- ✅ Read-aloud test: every variant fits one breath per sentence.
- ✅ Different from generic outbound: number-first / quote-first subjects, single-line openers, numbered or single-question CTAs, named competitors.
- ✅ Case study visible only where it strengthens (A-V1, A+D, D-V1) — never where it would seagull (Sandler #18).
- ✅ Unique mechanism visible per angle (list reactivation / paid funnel / outbound engine / coverage backstop).
- ✅ Pattern interrupt: documented per variant in the audit table above.
- ✅ Soft/valuable CTA: every CTA delivers a value-piece (count, preview list, 1-pager, audit) — none asks for the buyer's time.
- ✅ Sandler compliance: #2 (one ask), #5 (no preempt), #8 (appointment not sale), #12 (CTA = question), #18 (no seagulls), #27 (let them discover), #38 (real pain, 5th-Why hit per variant), #49 (no neediness — zero "would love to," "hope," "just checking in").
